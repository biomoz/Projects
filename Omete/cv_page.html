About Me

Who is Omer Celebi?

My hobbies are playing video games, fitness and programming. I'm interested in technology and games. Right now I'm trying to learn programming with all I can so that I can work as a programmer in the future.

The reason why I got interested in programming comes from my childhood. When I was a child I would play games or install things to my computer and one in a while there would be some errors accuring. I liked solving those kind of problems and fixing things by typing code into the command line or fixing files with other applications.

Right now I'm studying in the ICT faculty of Thomas More University with the specialization of Application Development. For my highschool, I graudated from ASO Sint Michielsinstituut Keerbergen in 2021.

During highschool, I had taken extra classes for programming. So I have some knowladge about programming. I have done a basic and small game project during highschool. Other than that I haven't done any projects before starting university. Right now I'm working on three projects:

While I try to code and create new things I'm still new to programming so I lack knowledge. But I'm eager to learn so I hope I will be succesful.
